Descrição

Você sabe o que são programas de computador? Programas de computador são construídos a partir de linguagens de programação. Aprender programação auxilia no raciocínio lógico e é um ótimo passatempo. Além disso, o mercado de trabalho está sempre a procura de profissionais com conhecimentos nesta área. Neste curso você aprenderá a construir seu primeiro programa de computador usando a linguagem Python.
