Descrição

Você sabe o que são programas de computador? Programas de computador são construídos a partir de linguagens de programação. Aprender programação auxilia no raciocínio lógico e é um ótimo passatempo. Neste curso você aprenderá a construir seu primeiro programa de computador usando a linguagem Python.
